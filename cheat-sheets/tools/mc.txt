Одна большая панель (можно использовать alt + t):
Listing format... -> Long file list

Темы хранятся в `/usr/share/mc/skins` и чтобы прописать тему по-умолчанию нужно
подправить файл `~/.config/mc/ini` атрибут `skin`, поставить туда название темы
из `/usr/share/mc`.

Если нужно изменить тему глобально, то в файл `default.ini` нужно вписать
название темы.

Для того чтобы изменить `mcedit` по-умолчанию на другой редактор, необходимо
открыть `F9` перейти в `Options` и в `Configurations`
убрать крестик с `Use internal edit`.

Действие над файлом:
Выделить файл, затем `F2 + @`, ввести команду.

Появилась возможно управлять `mc` с помощью `/etc/mc/mc.vim.keymap`.
Надо поменять `/etc/mc/mc.keymap` ссылку на `/etc/mc/mc.vim.keymap`.
